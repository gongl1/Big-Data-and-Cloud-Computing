Step 4. Establish similarity threshold and return near-duplicate records¶

Note: we are joining dataframe to itself to get near-duplicate pairs¶

Also note: you do not join on specific variables like keys or IDs. Instead, you join based on the similarity of feature vectors, which is determined using LSH algorithms.

These code use the MinHashLSH model to find pairs of documents with Jaccard similarities above a specified threshold (jaccard_distance). The resulting DataFrame (df_dups_text) contains pairs of similar documents along with their Jaccard distances and text content.

.filter("datasetA.id < datasetB.id"): This line filters out pairs of documents where the ID of the first document (datasetA.id) is less than the ID of the second document (datasetB.id). This ensures that each pair is only considered once, avoiding duplicate comparisons and self-comparisons.

a Jaccard distance of 0.3 means that pairs of sets with a Jaccard similarity (1 - Jaccard distance) greater than or equal to 0.7 will be considered similar.¶
